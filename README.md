# BaiduPano-unity-plugin

## Usage

In the unity3d LBS Game, pick a coordinate and show the real world scenery.

## Introduce

- iOS only.
- Add a UIKit view upon unityGLView.
- libstdc++.6.0.9.tbd removed in Xcode10.
